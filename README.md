# Simple React + TypeScript + SASS app (no boilerplate)

Commands:

1. Install dependencies

   npm install

2. Start dev server

   npm start

3. Build production bundle

   npm run build

Files:
- `webpack.config.js` – development + production config
- `babel.config.json` – Babel presets (env, react, typescript)
- `tsconfig.json` – TypeScript compiler options (noEmit)

This project uses Babel to transpile TypeScript and JSX and Sass for styling.
